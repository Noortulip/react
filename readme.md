# React Counter

A simple counter app built with React to practice core fundamentals: functional components, props, and the `useState` hook.

## Features
- Increment and decrement the counter value
- Reset button to set the counter back to 0
- Conditional rendering (e.g., disable decrement at 0, or show a message)
- Built with functional components and clean state management

## Tech Stack
- React
- JSX
- useState hook

## What I Learned
- Managing component state with `useState`
- Updating UI reactively based on state changes
- Passing props between components
- Handling conditional rendering based on state

## Getting Started

Clone the repo and install dependencies:

\`\`\`bash
git clone https://github.com/your-username/react-counter-app.git
cd react-counter-app
npm install
npm run dev
\`\`\`
